# Iris Flower Classification using Machine Learning

## Project Overview

This project is a beginner-level Machine Learning project that aims to classify Iris flowers into different species using Logistic Regression.

The project covers the basic workflow of a Machine Learning pipeline including data loading, exploratory data analysis, preprocessing, model training, and evaluation.

---

## Dataset

The dataset used in this project is the Iris Dataset.

The Iris dataset contains 150 samples of Iris flowers with four features:

- Sepal Length
- Sepal Width
- Petal Length
- Petal Width

The target variable contains three classes:

- Iris Setosa
- Iris Versicolor
- Iris Virginica

---

## Project Steps

The following steps were performed:

1. Import required Python libraries
2. Load the Iris dataset
3. Convert data into a Pandas DataFrame
4. Perform Exploratory Data Analysis (EDA)
5. Visualize data using charts
6. Split data into training and testing sets
7. Apply feature scaling
8. Train Logistic Regression model
9. Evaluate model performance
